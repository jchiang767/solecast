Data Sources

Switch sources with the Data Source picker in the sidebar. Each measures something different — use them together to separate real demand from hype.

SourceWhat it measuresProsCons
Google Trends Relative search interest (0–100), normalized across your keywords. Broadest reach; captures general public curiosity; supports regions. Aggressively rate-limits (falls back to sample data); relative-only, no absolute volume.
Wikipedia Daily pageviews for the matching article — people researching a brand/model. Free, reliable, never rate-limited; real absolute numbers; regions via language editions. Only covers terms with a wiki article; lags ~1–2 days; research intent ≠ buying intent.
Reddit Post volume weighted by engagement across sneaker/streetwear subreddits — community buzz. Earliest signal of hype; captures culture & sentiment before mainstream search. Requires your own API credentials (below); no geography; noisier, community-skewed.

Note: Related Queries is a Google-search feature, so it's only available on the Google source. Region Compare works on Google and Wikipedia (by language), not Reddit.

Reddit API Credentials

Reddit blocks anonymous access, so the Reddit source needs your own free app credentials. Create one at reddit.com/prefs/apps → create app → choose “script” → copy the client ID (under the app name) and the secret.

Stored locally in config.json on your machine — never sent anywhere except Reddit.
